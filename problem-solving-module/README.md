# 问题处理页面

这是一个基于React和TDesign组件库构建的问题处理页面，实现了问题记录、处理和评论功能。

## 功能特性

### 1. 问题处理
- 新增处理事项按钮
- 处理事项表单包含：处理事项、处理人、预计完成时间、实际完成时间
- 处理事项一旦创建不可修改、删除，只能新增

### 2. 处理记录
- 表格形式展示处理记录
- 按创建时间排序，由近到远
- 包含字段：创建时间、处理事项、处理人、预计完成时间、实际完成时间、处理结果、处理完成填写人
- 创建后自动发送企业微信+邮件提醒（功能模拟）
- 可填写处理结果（弹窗形式）

### 3. 评论功能
- 所有成员均可评论
- 按评论人+评论时间+评论内容展示
- 按评论时间排序，由近到远

## 技术栈

- React 18
- TDesign React组件库
- Day.js（日期处理）

## 运行说明

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 打开浏览器访问 http://localhost:3000

## 使用说明

1. 点击"新增处理事项"按钮，填写表单信息
2. 在处理记录表格中，对于未填写结果的事项，可点击"填写"按钮添加处理结果
3. 在评论区域输入评论内容，点击"提交评论"按钮添加评论

## 注意事项

- 本项目使用模拟数据，实际使用时需要连接真实API
- 用户信息和提醒功能为模拟实现，实际使用需要集成企业微信和邮件系统
